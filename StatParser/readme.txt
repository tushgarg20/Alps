to build StatParser in Windows: open StatParser.sln with MS Visual Studio 10

to build StatParser in Linux just type: make

to run StatParser in Linux you may need to set the LD_LIBRARY_PATH:
setenv LD_LIBRARY_PATH /p/gat/tools/boost/1.43.0/gcc4.3/lib64
